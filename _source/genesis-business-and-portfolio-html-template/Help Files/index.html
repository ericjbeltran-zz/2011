A) HTML Structure - top

This template is fixed layout with two columns. This is simple template and the html are commented well so you can easily make an update.

HERE IS THE HTML STRUCTURE:

-- LOGO AND NAVIGATION --

Logo is nested within a div with an id of "logo".

The top navigation (main menu) is nested within a ul with an id of "menu".

-- HOME PAGE SLIDER --

The slider area is nested within a div with an id of "slider".

you can add new img within div with an id of "slider" to add new slider.

-- HOME PAGE CONTENT AREA --

Home page content area is single column.

The intro text in the home page is nested within a div with a class of "middle-text".

The four column information in the home page is nested within an ul with a class of "four_column".

-- INNER PAGE CONTENT AREA --

The content area is divide by two column, in the left side is the main content area which is nested within a div with an id of "content" and the right side is the right sidebar area which is nested within a div with an id of "sideright".

Each sidebar item in the right side is nested within an li with class of "widget-container".

The content text under the "content" div is nested within a h1 tag with a class of "pagetitle"(for header) and p tag.

Pages -> Default page:

The four people information is nested within an ul with a class of "display-inline-block".

The four square with text "Client **." are nested within an ul with a class of "display-inline".


Blog page:

In the blog page, each blog post is nested within a div with a class of "post". You can add or change the content under this class and you can add new blog post item by adding new div with a class of "post" under the latest div.

-- BLOG DETAIL PAGE --
Blog detail content is nested within a div with an id of "singlepost".

About the author is nested within a div with an id of "authorbox". and related post is nested within a div with an id of "relatedbox".

Related Post is nested within a div with an id of "relatedbox". And Comments is nested within a div with an id of "comment"

--SLIDER PAGE --

Slider Comments in this page is nested within a div with a class of "ts-thinkboxslider".

Slider image in this page is nested within a div with an id of "slider-small".

-- JQUERY TAB PAGE --

Tabbing in this page is nested within a div with a class of "tabcontainer". and the tab itself is nested within an ul with a class "tabs". For the tab body, you can take a look at div with an id "tab-body"

-- PORTFOLIO PAGE --

Portfolio page content area is a single single column and it nested within a div with an id of "content".

The portfolio area is nested within an ul with id of "ts-display-pf-col-1" for portfolio1, "ts-display-pf-col-2" for portfolio2, "ts-display-pf-col-3" for portfolio3 and "ts-display-pf-col-4" for portfolio4.

And each portfolio item is nested within a li with a class name of its category, you can add an additional li tag to add more portfolio items.

Filter above portfolio is nested within div with id "filter". Each filter is nested in li.

-- CONTACT PAGE --

Contact page is divide by two column with googlemaps iframe on the sidebar.

The contact form is nested within a div with an id "contactform".

-- FOOTER AREA --

The five column area on the bottom is nested within a div with a class of "one_fifth".

The footer area is nested within a div with an id of "footer" .

B) CSS Files and Structure - top

These are the css files:
1) style.css (used for all pages, this is the general css file)
2) inner.css
3) prettyPhoto.css
4) nivo-slider.css

===========================================

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

1) style.css structure

/*** GLOBAL ***/

This is the global css for html, body, etc tags. Keep in mind, that these values might be overridden somewhere else in the file.

/*** END OF GLOBAL ***/

/*** DEFAULT ELEMENT STYLES ***/

This section contains some general styling, such as anchor tag colors, headings, list, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

/*** END OF DEFAULT ELEMENT STYLES ***/

/*** CONTAINER ***/

This is the main background of each page. You can change the main background image and color by change the styles in this section.

/*** END OF CONTAINER ***/

/*** TOP ***/

This section contains the styling for logo and top menu.

/*** END TOP ***/

/*** CONTENT ***/

This is the styling for all pages content area, divide the sidebar (right side) and main content(left side).

The content images style is also setup here: 

	.imgleft{ float:left; padding-right:15px; }
	 .imgright{ float:right; padding-left:15px; }

The right side sidebar styling also setup here.

/*** END OF CONTENT ***/

/*** FOOTER ***/

This section contains the styling for footer area.

/*** END OF FOOTER ***/

2)inner.css
